# 马房山

马房山的村庄具有独特机制。

1. 村庄如果持续处于玩家控制则可以为玩家提供行政点。
2. 村庄如果持续处于玩家控制将不断提高对玩家的忠诚度等级，忠诚度等级越高被对手夺取的难度就越大，能提供给玩家的奖励与开发手段也越多。
   当控制村庄时，可以慢慢等待村庄提高忠诚度，也通过花钱拉拢提高村庄的忠诚度。
3. 每个村庄拥有自己的**“村庄政策”**，若这个村庄成为了玩家帝国的核心领土，村庄政策将被纳入帝国的政策库。
4. 忠诚度等级在村庄易手后不会立即清零，而是缓慢降低。村庄对双方的忠诚度等级互相不影响。
   * 一级忠诚 **『军事控制』**：为占领玩家一个额外工人和资源采集加成。
   * 二级忠诚 **『羁縻统治』**：为占领玩家另一个额外工人，同时玩家在村庄附近的单位不消耗补给。
   * 三级忠诚 **『行政管辖』**：为占领玩家另一个额外工人，并允许在村庄上建造建筑。
   * 四级忠诚 **『核心领土』**：永久获得该村庄的独特政策，提供生产力加成，在村庄附近的单位战斗力加成。
  
5. 可以在 **『行政管辖』** 的村庄派出兵员。
