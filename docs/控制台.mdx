# 控制台

PC版点击<kbd>~</kbd>键开启，手机版在主界面设置中开启。

| 指令  | 参数   | 效果   |
|-------------- | -------------- | -------------- |
| cheat   | tomato     | 开启作弊（仅限单人）     |
| gold   | 数量     | 添加黄金     |
| food   | 数量     | 添加食物     |
| pdct   | 数量     | 添加生产力     |
| pop   | 数量     | 添加人口上限     |
| actp   | 数量     | 添加行政点数     |
| worker   | 数量     | 添加工人上限     |
| freeunit   | 士兵在下方槽位中的序号，数量     | 添加免费士兵     |
| freebuilding   | 建筑在建造面板中的序号，数量     | 添加免费建筑     |
| pol   | 政策id     | 添加政策     |
