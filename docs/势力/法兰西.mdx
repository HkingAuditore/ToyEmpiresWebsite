# 法兰西
    
## 独特效果

* <Highlight color="#4EC5F1">朕即国家</Highlight>抽取政策池由3个变为4个
  
* <Highlight color="#eb4c4c">骑士精神</Highlight>骑兵系士兵+20%体力值
  
* <Highlight color="#4EC5F1">三级会议</Highlight>开局获得政策**三级会议**
  
* <Highlight color="#4EC5F1">富农经济</Highlight>采集黄金的工人速度+30%。

## 独特政策

| 名称     | 效果                                       | 占用槽位 |
|--------|------------------------------------------|----------|
| 重商主义 | 无资源兑换损耗，但工人、士兵的生产速度-50%   | 2        |
| 炮兵改革 | 火炮的攻击力+30%,移动速度+30%，造价+100%    | 4        |
| 雇佣兵制 | 无需向士兵支付维护费，但士兵造价+50%        | 3        |
| 三级会议 | 敌人士兵越过战场中线后，工人采集的资源量+1。 | 3        |
